Escolha uma linguagem:

HTML

É a linguagem responsável por estruturar o conteúdo de páginas na web, organizando elementos como textos, imagens, botões e links.


Em andamento...

CSS

É a linguagem com foco em definir o visual das páginas web, controlando cores, tamanhos, fontes, espaçamentos e o layout geral.


Em andamento...

JavaScript

Linguagem voltada para adicionar interatividade e dinamismo às páginas web, permitindo reações a cliques, animações e mudanças em tempo real.


Em andamento...

C

Linguagem com foco em programação de baixo nível, ideal para desenvolvimento de sistemas operacionais, drivers e aplicações que exigem alto desempenho.


Em andamento...

C++

Linguagem com foco em desempenho e controle, usada principalmente no desenvolvimento de jogos, sistemas, aplicações pesadas e softwares que exigem acesso direto à memória, como engines gráficas.

C#

É uma linguagem criada com foco no desenvolvimento de aplicações modernas para Windows, além de ser amplamente usada em jogos com o motor Unity.

Lua

Linguagem leve e embutível, muito usada para criar scripts dentro de jogos e softwares, com foco em simplicidade e rapidez de execução.

Python

Linguagem com foco em simplicidade e legibilidade, amplamente usada para automação, ciência de dados, inteligência artificial e desenvolvimento web.

Java

É uma linguagem voltada para o desenvolvimento de aplicações multiplataforma, com forte presença em sistemas corporativos, aplicativos Android e ambientes que exigem portabilidade e segurança.

SQL

Linguagem usada exclusivamente para consultar e manipular bancos de dados relacionais. Indispensável no backend de aplicações web.

PHP

Apesar das críticas, ainda é amplamente usado no desenvolvimento web (WordPress, por exemplo). Simples, mas em declínio frente ao Node.js e outros.
