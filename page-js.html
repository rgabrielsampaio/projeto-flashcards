JavaScript

window.alert()

Exibe uma caixa de diálogo com uma mensagem e um botão "OK".

window.alert("Olá, mundo!");

window.confirm()

Exibe uma caixa de diálogo com uma mensagem e dois botões: "OK" e "Cancelar". Retorna true se "OK" for clicado, e false se "Cancelar" for clicado.

window.confirm("Você tem certeza?");

window.prompt()

Exibe uma caixa de diálogo que solicita ao usuário que insira um valor. Retorna o valor inserido ou null se "Cancelar" for clicado.

window.prompt("Digite seu nome:");

String -> Número

Com Number()

let idade = Number("25");

Com parseInt() ou parseFloat()

let idade = parseInt("25");
let preco = parseFloat("19.99");

Número -> String

Com String()

let idade = String(25);

Com toString()

let moedas = 50;
let texto = moedas.toString();
